# Crop and Weed Detection using Machine Learning

This project was developed as part of the internship program by upSkill Campus in collaboration with UniConverge Technologies Pvt Ltd.

## Project Description
The goal of this project is to build a machine learning model that can classify plant images as either crops or weeds.

## Technologies Used
- Python
- OpenCV
- NumPy
- Scikit-learn

## Steps to Run

1. Install dependencies

pip install -r requirements.txt

2. Train the model

python crop_weed_detection.py

3. Run prediction

python predict.py
